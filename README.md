# Cakes Store

Cakes Store — это простой фронтенд-прототип каталога тортов. Проект демонстрирует, как может выглядеть интерфейс небольшого интернет-магазина: карточки товаров, минимальная навигация и базовая логика взаимодействия на стороне клиента.

## Технологии и используемые библиотеки

-   React — построение пользовательского интерфейса
-   React Router — маршрутизация между страницами
-   React Redux — управление состоянием приложения
-   Mock Service Worker (MSW) — эмуляция API и подмена сетевых запросов для удобной разработки
-   Yup — валидация форм и пользовательских данных
-   Vite — сборщик и локальный дев-сервер
-   HTML5 / CSS3
-   JavaScript
